Cloud Sync

Backs up your routines and history to a private GitHub Gist after every workout. Restores automatically if local data is lost. Requires a GitHub Personal Access Token with gist scope only.

Setup
  1. github.com/settings/tokens — generate a Classic token with only the gist scope
  2. gist.github.com — create a new secret gist with filename lift.json and content {}
  3. Copy the gist ID from its URL (the long hex string)
  4. Paste both above and tap Save
